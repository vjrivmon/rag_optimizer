# Plantilla de Seccion - TFG Chatbot RAG DNI

## Estructura de Escritura (Patron Joan/Rosa)

Cada seccion del Marco Teorico debe seguir este patron:

### 1. Contexto Introductorio (1-2 parrafos)
- Por que es relevante este tema
- Problema que aborda
- Conexion con el proyecto

### 2. Definicion Clara (1 parrafo)
- Definicion academica con referencia [n]
- Origen del termino si aplica

### 3. Ampliacion Conceptual (2-3 parrafos)
- Subtipos o variantes
- Teorias relacionadas
- Evolucion historica si es relevante

### 4. Ejemplos Practicos (1-2 parrafos)
- Casos reales de uso
- Implementaciones conocidas
- Datos o estadisticas

### 5. Conexion con el Proyecto (1 parrafo)
- Como aplica al Chatbot RAG DNI
- Justificacion de elecciones tecnicas
